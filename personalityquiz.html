Personality Quiz

Personality Quiz

  • Sep 3rd, 2021
  • Advanced

Materials

  • Platform to code on: Glitch, Text Editor, Atom, Etc...
  • Any materials you'd like to plan out your final product

Step 1: Plan out what you'd like on your personality quiz.

You can look up examples on websites like Buzzfeed. Write done what questions you'd like to include, images, and other pieces of content.

Step 2: Get on a coding platform of your choice and start coding the basic htmls and javascript needed.

For references of what the language, refer to:

  • CodePen | All
  • W3Schools | HTML

Step 3: Add CSS code to customize and design aspects to it.

Check out any of the following links to learn more about CSS Code

  • CodePen
  • W3Schools

Step 4: Test it!

Always make sure you are testing your code to ensure it is working correctly and allow easy debugging if there were to be any bugs.

Step 5: Share it!
